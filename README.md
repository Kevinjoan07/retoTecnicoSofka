# retoTecnicoSofka


Reto tecnico el cual consiste en realizar  pruebas a la Api  https://petstore.swagger.io/

se realizaron las siguiente pruebas 
<ul>
  <li>Consultar una orden de compra</li>
  <li>Buscar la orden de compra creada</li>
  <li>Verificar el inventario de ventas</li>
  <li>Eliminar una orden de compra</li>
</ul>

estas pruebas automatizadas fueron realizadas con el  framework de pruebas Playwright

# Ejecucion
Para ejecutar todos los casos de prueba  usar el comando <code>npx playwright test</code> <br>
Para la ejecucion  y visualizacion del reporte ejecutar el comando  <code>npx playwright show-report</code>
